# DSX Code — 基于 DeepSeek V4 的下一代终端智能编程助手

DSX Code (`dsx`) 是一款采用 Rust 语言编写，专为极客与开发者打造的下一代终端 AI 编程助手。它不仅仅是一个简单的 API 封装，而是一个能够在本地自主运行的智能体环境（Agent Runtime），集成了自动代码修改、命令执行、基于 SQLite 的会话持久化，以及极具未来科技感的 Tron 风格终端 TUI 控制面板。

---

## 🚀 核心产品特性

1. **Tron 风格交互式 TUI:**
   * 采用超高响应速度的 Ratatui 渲染。
   * 支持可折叠的双侧边栏：左侧为 `📁 WORKSPACE` 工作区文件树，右侧为实时流式传输的 `🧠 THOUGHT PROCESS` 深度推理思维面板。
   * 嵌套式的会话日志，自动折行处理，且具备格式化的工具调用时间轴。

2. **高容错度的智能路由分类器:**
   * 采用极低成本的 `deepseek-v4-flash`（Non-Thinking 模式）配合 JSON Schema 格式验证，自主分析任务复杂度并完成模型分类。
   * 在发生网络超时或断网时，可无缝且安全地自动降级到本地启发式分类规则，确保任务执行的绝对连续性。

3. **极度精准安全的 3 阶 Patch 引擎:**
   * 拒绝让模型大范围覆写文件，而是提出 SEARCH/REPLACE 的 Patch 补丁修改方案。
   * 具备 3 阶高精度对齐机制：精确匹配、忽略空白符匹配、以及保持原有缩进对齐匹配。
   * 能够做到不破坏源文件的任何原有格式特征（缩进、换行、编码）。

4. **安全首要的交互式授权网关 (Security Gate):**
   * 在 `Ask` 模式下，当模型尝试运行中高风险工具或 shell 命令行时，会自动挂起 Agent 的后台运行。
   * 终端将呈现醒目的红色安全警示块，等待用户输入 `[Y]` 确认运行或 `[N]` 予以拦截拒绝。

5. **一键撤销与 Git 回滚机制 (`Ctrl+U`):**
   * 在模型实际触碰任何代码文件前，系统会自动在 Git 仓库中创建一个 `DSX checkpoint:` 临时恢复点。
   * 按下 **`Ctrl+U`**，即可一键重置当前工作区的所有修改，且侧边栏的文件树会自动刷新同步。

6. **交互式系统配置中心 (`Ctrl+S`):**
   * 按下 **`Ctrl+S`**，主控制面板将切换为配置中心，支持通过方向键修改保护模式、切换硬/快模型、更改系统语言、调整 API 服务端基准地址，以及一键清空会话历史。

7. **多语言国际化支持 (i18n):**
   * 界面及交互式状态完全支持 4 国语言：中文、俄文、哈萨克文、以及英文。

## ⌨️ 全局快捷键指南

| 快捷键 | 对应动作 |
| :--- | :--- |
| `Ctrl+S` | 开启 / 关闭交互式系统设置配置面板 |
| `Ctrl+T` | 开启 / 隐藏左侧工作区文件资源浏览器 |
| `Ctrl+D` | 开启 / 关闭当前代码差异对比（Workspace Diff）分析面板 |
| `Ctrl+U` | 一键撤销（一键回滚代码至最近的自动备份点） |
| `Ctrl+C` | 安全退出当前会话 |
| `Esc` | 清空当前输入框文字 / 退出当前活动面板 |
| `Enter` | 提交任务至 ИИ 核心 / 确认菜单执行项 |

## 🛠️ CLI 安装与配置说明

### 配置 API 密钥
```bash
export DEEPSEEK_API_KEY="sk-..."
```

### 安装
直接通过 Cargo 将本项目本地源码编译并全局安装至您的环境 `PATH` 路径中 (`~/.cargo/bin/dsx`):
```bash
cargo install --path . --force
```

### 命令列表
```bash
# 启动交互式 TUI 工作空间 (默认)
dsx

# 以极速 YOLO 自动确认模式启动 TUI
dsx --mode yolo

# 仅请求生成一份高维度架构设计方案
dsx plan "将 auth 模块重构为异步 reqwest 架构"

# 直接触发点对点修改，无需打开 TUI 界面
dsx edit "修复 memory_store.rs 文件中的栈溢出漏洞"

# 调出本地 SQLite 持久化保存的历史会话列表
dsx workspace list

# 重新载入之前的某一历史会话，恢复其全部上下文、Token 与费用统计
dsx workspace resume <会话_id>
```
