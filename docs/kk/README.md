# DSX Code — DeepSeek негізіндегі Консольдік ИИ Кодинг Агенті

DSX Code (`dsx`) — бұл DeepSeek V4 негізіндегі, Rust тілінде жазылған жаңа буынның консольдік ИИ кодинг ассистенті. Ол файлдарды басқаратын, шелл командаларын орындайтын, SQLite арқылы сессияларды сақтайтын және Tron стиліндегі өте әдемі TUI интерфейсін ұсынатын автономды жергілікті орта болып табылады.

---

## 🚀 Негізгі мүмкіндіктер

1. **Интерактивті TUI интерфейсі (Tron стилі):**
   * Бүйірлік панельдер: файлдар ағашы `📁 WORKSPACE` (сол жақта) және модельдің ойлау ағыны `🧠 THOUGHT PROCESS` (оң жақта), ол жауап беру басталғанда автоматты түрде жабылады.
   * Мәтінді автоматты түрде тасымалдайтын чат және құралдарды шақыру логтары.

2. **Ақауларға төзімді ИИ-роутер:**
   * Тапсырманың күрделілігін автоматты түрде бағалайды және оны `deepseek-v4-flash` (Non-Thinking) моделіне бағыттайды.
   * Желілік кідірістер кезінде жергілікті эвристикалық классификаторға автоматты түрде ауысады.

3. **Қорғалған көпдеңгейлі патч-қозғалтқыш:**
   * Код блоктарын (SEARCH/REPLACE) 3 қатаң деңгейде хирургиялық түрде өзгертеді: Дәл сәйкестік, бос орындарға сезімтал еместік және түпнұсқалық шегіністерді (отступ) сақтау.
   * Файлдың бастапқы пішімделуін толық сақтайды.

4. **Интерактивті қауіпсіздік шлюзі (Security Gateway):**
   * `Ask` режимінде қауіпті құралдар шақырылғанда агент жұмысын уақытша тоқтатады.
   * Терминал командаларын орындау алдында сізден растауды (`Y`/`N`) сұрайды.

5. **Бір рет басу арқылы өзгертулерді қайтару (Undo / Rollback):**
   * Файлдарды өзгертпес бұрын Git-те автоматты түрде чекпоинт коммитін жасайды.
   * `Ctrl+U` пернесін басу жобаны соңғы ИИ өңдеуіне дейінгі бастапқы күйіне бірден қайтарады.

6. **Интерактивті баптаулар (`Ctrl+S`):**
   * TUI ішінде қорғаныс режимдерін, белсенді модельді, интерфейс тілін (i18n), кастомдық API эндпоинттерін және авторизация кілттерін реттеңіз.

7. **Көптілді локализация (i18n):**
   * Интерфейс толығымен 4 тілге аударылған: Қазақша, Орысша, Ағылшынша және Қытайша.

---

## ⌨️ Ыстық пернелер

| Перне | Әрекет |
| :--- | :--- |
| `Ctrl+S` | Жүйелік баптаулар конфигураторын ашу/жабу |
| `Ctrl+T` | Жұмыс аймағының файлдар ағашын көрсету/жасыру |
| `Ctrl+D` | Ағымдағы өзгерістерді көру (Workspace Diff) |
| `Ctrl+U` | Өзгерістерді соңғы чекпоинтке қайтару (Undo) |
| `Ctrl+C` | Сессиядан қауіпсіз шығу |
| `Esc` | Енгізу өрісін тазарту / белсенді панельді жабу |
| `Enter` | Тапсырманы чатқа жіберу / мәзір әрекетін растау |

## 🛠️ Орнату және Іске қосу

### API кілтін реттеу
```bash
export DEEPSEEK_API_KEY="sk-..."
```

### Орнату
Бинарлық файлды Cargo PATH (`~/.cargo/bin/dsx`) ішіне жаһандық деңгейде орнатыңыз:
```bash
cargo install --path . --force
```

### Командалар
```bash
# Интерактивті TUI іске қосу
dsx

# Максималды авто-режимде іске қосу (Yolo)
dsx --mode yolo

# Сәулеттік өзгерістердің жылдам жоспарын жасау
dsx plan "auth модулін асинхронды reqwest-ке ауыстыру"

# TUI ашпай-ақ нүктелік өзгеріс енгізу
dsx edit "memory_store.rs файлындағы толып кету қатесін түзету"

# SQLite-тегі сессиялар тарихын көру
dsx workspace list

# Алдыңғы сессияны барлық чаттармен және токендермен қалпына келтіру
dsx workspace resume <сессия_id>
```
