# DSX Code — ИИ-Агент для Кодинга в Терминале на базе DeepSeek

DSX Code (`dsx`) — это консольный ИИ-ассистент нового поколения, написанный на Rust и работающий на базе DeepSeek V4. Он действует как автономная локальная среда выполнения, управляющая файлами, выполняющая команды терминала, ведущая персистентность сессий через SQLite и предоставляющая невероятно красивый TUI-интерфейс в стиле Tron.

---

## 🚀 Ключевые возможности

1. **Интерактивный TUI-интерфейс (Стиль Tron):**
   * Выдвижные боковые панели: дерево файлов `📁 WORKSPACE` (слева) и живой поток размышлений модели `🧠 THOUGHT PROCESS` (справа), который уезжает при начале вывода ответа.
   * Форматированные логи чата с автопереносом строк и красивый лог вызовов инструментов.

2. **Отказоустойчивый ИИ-роутер задач:**
   * Автоматически оценивает сложность задачи и направляет её через дешевый запрос к `deepseek-v4-flash` (Non-Thinking) с валидацией JSON-ответа.
   * При сетевых задержках бесшовно переключается на локальный эвристический классификатор.

3. **Защищенный многоуровневый патч-движок:**
   * Хирургически заменяет блоки кода (SEARCH/REPLACE) в 3 строгих тира: Точное совпадение, нечувствительность к пробелам и сохранение оригинальной индентации (отступов).
   * Полностью сохраняет исходное форматирование файла.

4. **Интерактивный шлюз безопасности (Security Gateway):**
   * Приостанавливает выполнение агента при вызове средних/опасных инструментов в режиме `Ask`.
   * Запрашивает ваше подтверждение (`Y`/`N`) перед любым запуском команд шелла.

5. **Откат изменений в один клик (Undo / Rollback):**
   * Автоматически делает коммит-чекпоинт в Git перед любым изменением файлов.
   * Нажатие `Ctrl+U` мгновенно откатывает проект к исходному состоянию до последней правки ИИ.

6. **Интерактивные настройки (`Ctrl+S`):**
   * Прямо в TUI настраивайте режимы защиты, активную модель, язык интерфейса (i18n), кастомные API эндпоинты и ключи авторизации.

7. **Мультиязычная локализация (i18n):**
   * Полностью переведенный интерфейс на 4 языках: Русский, Казахский, Английский и Китайский.

## ⌨️ Горячие клавиши

| Клавиша | Действие |
| :--- | :--- |
| `Ctrl+S` | Открыть/закрыть Системный конфигуратор настроек |
| `Ctrl+T` | Скрыть/показать Дерево файлов рабочей области |
| `Ctrl+D` | Показать текущие измененные диффы (Workspace Diff) |
| `Ctrl+U` | Откатить изменения к последнему чекпоинту (Undo) |
| `Ctrl+C` | Безопасный выход из сессии |
| `Esc` | Очистить поле ввода / закрыть активную панель |
| `Enter` | Отправить задачу в чат / подтвердить действие меню |

## 🛠️ Установка и Запуск

### Настройка API-ключа
```bash
export DEEPSEEK_API_KEY="sk-..."
```

### Установка
Соберите и установите бинарник глобально в Cargo PATH (`~/.cargo/bin/dsx`):
```bash
cargo install --path . --force
```

### Подкоманды
```bash
# Запуск интерактивного TUI
dsx

# Запуск в максимальном авто-режиме (Yolo)
dsx --mode yolo

# Сделать быстрый план архитектурных изменений
dsx plan "Перевести модуль auth на асинхронный reqwest"

# Внести точечное изменение без открытия TUI
dsx edit "Исправить ошибку переполнения в memory_store.rs"

# Посмотреть историю сессий в SQLite
dsx workspace list

# Возобновить прошлую сессию со всеми чатами и токенами
dsx workspace resume <id_сессии>
```
